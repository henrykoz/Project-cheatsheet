Basic Terminal Commands

Command Description Example
ls list directory contents This command will list the contents of the current directory. For example:
$ ls
Desktop Documents Downloads
pwd print working directory This command will print the current working directory. For example:
$ pwd
/home/user/Desktop
cd change directory This command will change the current working directory. For example, to change to the "Documents" directory:
$ cd Documents
mkdir make a new directory This command will create a new directory with the specified name. For example, to create a new directory named "new_folder":
$ mkdir new_folder
rm remove a file or directory This command will remove a file or directory. For example, to remove a file named "old_file.txt":
$ rm old_file.txt
mv move or rename a file or directory This command will move or rename a file or directory. For example, to move a file named "file.txt" to the "Documents" directory:
$ mv file.txt Documents/
cp copy a file or directory This command will copy a file or directory. For example, to copy a file named "file.txt" to a new file named "file_copy.txt":
$ cp file.txt file_copy.txt
touch create a new file This command will create a new file with the specified name. For example, to create a new file named "new_file.txt":
$ touch new_file.txt
cat display the contents of a file This command will display the contents of a file. For example, to display the contents of a file named "file.txt":
$ cat file.txt
grep search for a pattern in a file This command will search for a pattern in a file. For example, to search for the pattern "apple" in a file named "fruit.txt":
$ grep apple fruit.txt
chmod change file permissions This command will change the permissions of a file or directory. For example, to give read, write, and execute permissions to the owner of a file named "file.txt":
$ chmod 700 file.txt
sudo execute a command with superuser privileges This command will execute a command with superuser privileges. For example, to install a package using a package manager:
$ sudo apt-get install package_name
man display the manual page for a command This command will display the manual page for a command. For example, to display the manual page for the "ls" command:
$ man ls
ps display a snapshot of the current processes $ ps
top display system resource usage and processes $ top
ssh connect to a remote machine over SSH This command will connect to a remote machine over SSH. For example, to connect to a machine with IP address "192.168.1.100":
$ ls -l
scp copy files securely over SSH This command will copy files securely over SSH. For example, to copy a file named "file.txt" to a remote machine with IP address "192.168.1.100":
$ scp file.txt username@192.168.1.100:/path/to/destination
zip compress files into a zip archive To create a zip archive called my_files.zip containing the files file1.txt, file2.txt, and file3.txt in the current directory:
$ zip my_files.zip file1.txt file2.txt file3.txt
unzip extract files from a zip archive To extract the contents of a zip archive called my_files.zip in the current directory:
$ unzip my_files.zip
wget download files from the internet To download a file called example_file.txt from a website:
$ wget https://example.com/example_file.txt
